== Description ==
Display Wordpress visits statistics. nStatistics plugin make statistics for unique visitors and pageviews, browsers used,
referer or search engine, keywords. Bots functions can be activated or dezactivated and log ip, crawler name and pageviews.
Stats are displayed in today graphs, date range graphs and by hour. All graphs have zoom function.
in visual graphs with zoom function. 


== Installation ==
**Upgrade**
1. First deactivate nStatistics plugin. 
2.Remove the `nstatistics` directory

**Install**
1. Unzip the `nstatistics.zip` file. 
2. Upload the the `nstatistics` folder (not just the files in it!) to your `wp-contents/plugins` folder.

**Activate**
1. From the Plugin page in Dashboard, find and activate nStatistics plugin

== How to uninstall Wordpress Cleanup ==
1. Deactivate plugin nStatistics
2. Delete folder nstatistics from wp-content\plugins\
